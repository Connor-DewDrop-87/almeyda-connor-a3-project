# almeyda-connor-a3-project
#### Premise:
Pawngger, how to play. The rules are similar to regular chess with these changes. 
You control a blue pawn, who must avoid the white pieces, lest they be executed. Every white piece moves in a set pattern. 
The Goal is reach the other side of the board and become a queen. 
#### Controls: 
Press Space to move forward
Press Enter to Reset if the game has been won or lost
While on the win screen, you can move using the W, A, S, and D keys.
